# وحدة الترجمة

في WindowTranslator، يمكنك اختيار واستخدام عدة وحدات ترجمة.  
كل وحدة لها خصائصها، ومن خلال اختيار الوحدة المناسبة حسب الاستخدام، يمكنك استخدام الترجمة بشكل أكثر راحة.

## Bergamot ![افتراضي](https://img.shields.io/badge/افتراضي-brightgreen)

وحدة ترجمة آلية تعمل دون اتصال.

### المزايا
- **مجاني تماماً**: بدون أي رسوم
- **بدون حد للترجمة**: ترجم بقدر ما تريد
- **سريع**: المعالجة المحلية للترجمة السريعة
- **الخصوصية**: لا حاجة لاتصال بالإنترنت، البيانات لا تُرسل للخارج
- **الاستقرار**: لا يتأثر بالشبكة

### العيوب
- **دقة الترجمة**: دقة الترجمة أقل مقارنة بخدمات السحابة
- **استخدام الذاكرة**: يستخدم كمية معينة من الذاكرة لمعالجة الترجمة
- **اللغات المدعومة**: يتم دعم بعض أزواج اللغات فقط

### سيناريوهات الاستخدام الموصى بها
- عندما تريد الاستخدام مجاناً
- الاستخدام في بيئة غير متصلة
- عندما تكون الخصوصية مهمة
- عندما تقوم بترجمات متكررة

---

## ترجمة Google

وحدة ترجمة تستخدم خدمة ترجمة Google.

### المزايا
- **مجاني تماماً**: يمكن استخدامه بدون مفتاح API
- **دعم متعدد اللغات**: يدعم العديد من أزواج اللغات
- **سهل**: لا حاجة لإعداد خاص

### العيوب
- **حد الترجمة**: حد على عدد الأحرف التي يمكن ترجمتها يومياً
- **دقة الترجمة**: قد تكون الدقة أقل مقارنة بالخدمات المدفوعة الأخرى
- **السرعة**: يتأثر بالشبكة
- **الاستقرار**: قد يصبح غير متاح فجأة بسبب قيود الاستخدام

### سيناريوهات الاستخدام الموصى بها
- الاستخدام منخفض التردد
- عندما تريد البدء فوراً
- عندما تريد ترجمة أزواج لغات متنوعة

---

## DeepL

وحدة تستخدم خدمة ترجمة DeepL، المعروفة بالترجمات عالية الجودة.

### المزايا
- **دقة عالية**: ترجمات طبيعية وعالية الجودة
- **طبقة مجانية كبيرة**: حتى 500,000 حرف مجاناً شهرياً (API مجاني)
- **سريع**: معالجة ترجمة سريعة
- **دعم المسرد**: حافظ على اتساق الترجمة مع المسارد

### العيوب
- **التسجيل في API مطلوب**: التسجيل في DeepL API وإعداد مفتاح API مطلوب
- **حد مجاني**: إذا تم تجاوز الطبقة المجانية، فإن الترقية إلى خطة مدفوعة مطلوبة
- **اللغات المدعومة**: اللغات المدعومة أكثر محدودية من Google

### سيناريوهات الاستخدام الموصى بها
- عندما تبحث عن ترجمات عالية الجودة
- الاستخدام متوسط التردد

---

## Google AI (Gemini)

وحدة ترجمة تستخدم أحدث تقنيات الذكاء الاصطناعي من Google.

### المزايا
- **أعلى دقة**: ترجمات عالية الجودة جداً تفهم السياق
- **المرونة**: تخصيص المطالبات لتعديل أسلوب الترجمة
- **دعم المسرد**: حافظ على اتساق الترجمة مع المسارد

### العيوب
- **مفتاح API مطلوب**: يجب الحصول على مفتاح API وإعداده من Google AI Studio
- **الدفع حسب الاستخدام**: رسوم بناءً على الاستخدام (لكن مبلغ صغير)
- **السرعة**: وقت معالجة أطول لأنه مبني على LLM

### سيناريوهات الاستخدام الموصى بها
- عندما تبحث عن ترجمات بأعلى جودة
- عندما يكون أسلوب ترجمة مخصص مطلوباً
- عندما تكون الترجمات الحساسة للسياق مطلوبة

---

## ChatGPT API (أو LLM محلي)

وحدة ترجمة تستخدم ChatGPT API أو LLM محلي.

### المزايا
- **أعلى دقة**: ترجمات عالية الجودة بواسطة نماذج اللغة الكبيرة
- **المرونة**: تخصيص المطالبات لتعديل أسلوب الترجمة
- **دعم المسرد**: حافظ على اتساق الترجمة مع المسارد
- **دعم LLM محلي**: إمكانية استخدام خادم LLM الخاص بك

### العيوب
- **مفتاح API مطلوب**: إعداد مفتاح API لكل خدمة مطلوب (باستثناء LLM المحلي)
- **الدفع حسب الاستخدام**: رسوم بناءً على الاستخدام (باستثناء LLM المحلي)
- **السرعة**: وقت معالجة أطول
- **متطلبات LLM المحلي**: كمبيوتر عالي الأداء مطلوب لتشغيل LLM الخاص بك

### سيناريوهات الاستخدام الموصى بها
- عندما تبحث عن ترجمات بأعلى جودة
- عندما يكون أسلوب ترجمة مخصص مطلوباً
- عندما تريد ترجمات عالية الجودة مع تقدير الخصوصية (LLM محلي)

---

## PLaMo

وحدة ترجمة تستخدم LLM محلي متخصص في اليابانية.

### المزايا
- **متخصص في اليابانية**: محسن للترجمات اليابانية
- **مجاني تماماً**: نموذج مفتوح المصدر بدون رسوم
- **الخصوصية**: يعمل محلياً، البيانات لا تُرسل للخارج
- **غير متصل**: لا حاجة لاتصال بالإنترنت

### العيوب
- **متطلبات أداء عالي**: كمبيوتر عالي الأداء مع GPU مطلوب
- **استخدام الذاكرة**: كمية كبيرة من الذاكرة مطلوبة (8 جيجابايت أو أكثر موصى به)
- **السرعة**: وقت معالجة أطول بدون GPU

### سيناريوهات الاستخدام الموصى بها
- عندما تمتلك كمبيوتر عالي الأداء
- عندما تكون الخصوصية هي الأكثر أهمية
- عندما تقدر جودة الترجمة اليابانية
