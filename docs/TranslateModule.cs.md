# Překladové moduly

WindowTranslator umožňuje vybrat z několika překladových modulů.  
Každý modul má své vlastnosti a výběr vhodného modulu pro váš případ použití umožní pohodlnější používání překladu.

## Bergamot ![Výchozí](https://img.shields.io/badge/Výchozí-brightgreen)

Modul strojového překladu fungující offline.

### Výhody
- **Zcela zdarma**: Bez jakýchkoli poplatků
- **Bez omezení překladu**: Můžete překládat kolikrát chcete
- **Rychlý**: Překlad je rychlý, protože se zpracovává lokálně
- **Soukromí**: Nevyžaduje připojení k internetu, data se neodesílají ven
- **Stabilita**: Není ovlivněn síťovými podmínkami

### Nevýhody
- **Přesnost překladu**: Nižší přesnost překladu ve srovnání s cloudovými službami
- **Využití paměti**: Spotřebovává určité množství paměti pro zpracování překladu
- **Podpora jazyků**: Podporovány jsou pouze některé jazykové páry

### Doporučené případy použití
- Chcete-li používat zdarma
- Pro použití v offline prostředí
- Pokud je důležité soukromí
- Při častém překládání

## Google Translate

Překladový modul využívající překladatelskou službu Google.

### Výhody
- **Zcela zdarma**: Lze používat bez klíče API
- **Vícejazyčná podpora**: Podporuje mnoho jazykových párů
- **Snadné**: Nevyžaduje speciální nastavení

### Nevýhody
- **Limity překladu**: Omezený počet znaků, které lze přeložit za den
- **Přesnost překladu**: Může být méně přesný ve srovnání s jinými placenými službami
- **Rychlost**: Závisí na síťových podmínkách
- **Stabilita**: Může náhle přestat být dostupný kvůli omezením používání

### Doporučené případy použití
- Příležitostné používání
- Chcete-li začít okamžitě
- Chcete-li překládat různé jazykové páry

## DeepL

Překladový modul využívající překladatelskou službu DeepL.

### Výhody
- **Vysoká přesnost překladu**: Vysoká kvalita překladů
- **Velká bezplatná kvóta**: Velkorysý bezplatný limit
- **Rychlost**: Rychlý překlad

### Nevýhody
- **Vyžadován klíč API**: Vyžaduje registraci a získání klíče API
- **Omezení**: Bezplatný plán má limity počtu znaků

### Doporučené případy použití
- Potřebujete-li vysoce kvalitní překlady
- Při častém překládání v rámci limitů bezplatného plánu

### Výhody

- **Kontextový překlad**: Chápe kontext a překládá přirozeně

### Nevýhody
- **Vyžadován klíč API**: Vyžaduje registraci a získání klíče API
- **Náklady**: Vyžaduje malý poplatek

### Doporučené případy použití
- Potřebujete-li nejvyšší kvalitu překladů
- Pokud je důležitý kontextový překlad

## Lokální LLM

Překladový modul využívající lokálně spuštěný jazykový model.

### Výhody
- **Soukromí**: Data se neodesílají ven
- **Bez nákladů na službu**: Samotná služba je po nastavení zdarma

### Nevýhody
- **Vyžaduje výkonný počítač**: Vyžaduje výkonný počítač
- **Nastavení**: Vyžaduje pokročilou konfiguraci

### Doporučené případy použití
- Pokud je soukromí nejdůležitější
- Máte-li výkonný počítač
- Chcete-li mít plnou kontrolu nad překladem
