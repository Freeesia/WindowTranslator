## İndirme
### Microsoft Store Sürümü ![Önerilen](https://img.shields.io/badge/Önerilen-brightgreen)

[Microsoft Store](https://apps.microsoft.com/detail/9pjd2fdzqxm3?referrer=appbadge&mode=direct)'dan yükleyin.
.NET yüklü olmayan ortamlarda bile çalışır.

### Yükleyici Sürümü

[GitHub sürümler sayfasından](https://github.com/Freeesia/WindowTranslator/releases/latest) `WindowTranslator-(sürüm).msi` dosyasını indirin ve çalıştırarak yükleyin.

### Taşınabilir Sürüm

[GitHub sürümler sayfasından](https://github.com/Freeesia/WindowTranslator/releases/latest) zip dosyasını indirin ve istediğiniz bir klasöre çıkarın.
- `WindowTranslator-(sürüm).zip` : .NET ortamı gerektirir
- `WindowTranslator-full-(sürüm).zip` : .NET'ten bağımsız

## Nasıl Kullanılır

### Bergamot ![Varsayılan](https://img.shields.io/badge/Varsayılan-brightgreen)

1. `WindowTranslator.exe` dosyasını başlatın ve çevir düğmesine basın.
   ![Çevir Düğmesi](images/translate.png)
2. Çevirmek istediğiniz uygulamanın penceresini seçin ve "OK" düğmesine basın.
   ![Pencere Seçimi](images/select.png)
