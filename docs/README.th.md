## ดาวน์โหลด
### เวอร์ชัน Microsoft Store ![แนะนำ](https://img.shields.io/badge/แนะนำ-brightgreen)

ติดตั้งจาก [Microsoft Store](https://apps.microsoft.com/detail/9pjd2fdzqxm3?referrer=appbadge&mode=direct)
ทำงานได้แม้ในสภาพแวดล้อมที่ไม่ได้ติดตั้ง .NET

### เวอร์ชันตัวติดตั้ง

ดาวน์โหลด `WindowTranslator-(เวอร์ชัน).msi` จาก[หน้ารีลีสของ GitHub](https://github.com/Freeesia/WindowTranslator/releases/latest) และรันเพื่อติดตั้ง

### เวอร์ชันพกพา

ดาวน์โหลดไฟล์ zip จาก[หน้ารีลีสของ GitHub](https://github.com/Freeesia/WindowTranslator/releases/latest) และแตกไฟล์ไปยังโฟลเดอร์ใดก็ได้
- `WindowTranslator-(เวอร์ชัน).zip` : ต้องการสภาพแวดล้อม .NET
- `WindowTranslator-full-(เวอร์ชัน).zip` : ไม่ขึ้นกับ .NET

## วิธีการใช้งาน

### Bergamot ![ค่าเริ่มต้น](https://img.shields.io/badge/ค่าเริ่มต้น-brightgreen)

1. เปิด `WindowTranslator.exe` และคลิกปุ่มแปล
   ![ปุ่มแปล](images/translate.png)
2. เลือกหน้าต่างของแอปพลิเคชันที่คุณต้องการแปลและคลิกปุ่ม "OK"
   ![การเลือกหน้าต่าง](images/select.png)
