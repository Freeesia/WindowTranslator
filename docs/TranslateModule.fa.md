# ماژول ترجمه

در WindowTranslator، می‌توانید چندین ماژول ترجمه را انتخاب و استفاده کنید.  
هر ماژول ویژگی‌های خاص خود را دارد و با انتخاب ماژول مناسب بر اساس کاربرد، می‌توانید از ترجمه به صورت راحت‌تری استفاده کنید.

## Bergamot ![پیش‌فرض](https://img.shields.io/badge/پیش‌فرض-brightgreen)

ماژول ترجمه ماشینی که بدون اتصال به اینترنت کار می‌کند.

### مزایا
- **کاملاً رایگان**: بدون هیچ هزینه‌ای
- **بدون محدودیت ترجمه**: هر مقدار که بخواهید ترجمه کنید
- **سریع**: پردازش محلی برای ترجمه سریع
- **حریم خصوصی**: نیاز به اتصال به اینترنت ندارد، داده‌ها به خارج ارسال نمی‌شوند
- **پایداری**: تحت تأثیر شبکه نیست

### معایب
- **دقت ترجمه**: دقت ترجمه در مقایسه با سرویس‌های ابری کمتر است
- **استفاده از حافظه**: مقدار مشخصی حافظه برای پردازش ترجمه استفاده می‌کند
- **زبان‌های پشتیبانی‌شده**: تنها برخی جفت‌های زبانی پشتیبانی می‌شوند

### سناریوهای استفاده توصیه‌شده
- وقتی می‌خواهید رایگان استفاده کنید
- استفاده در محیط آفلاین
- وقتی حریم خصوصی مهم است
- وقتی ترجمه‌های مکرر انجام می‌دهید

---

## ترجمه Google

ماژول ترجمه‌ای که از سرویس ترجمه Google استفاده می‌کند.

### مزایا
- **کاملاً رایگان**: می‌توان بدون کلید API استفاده کرد
- **پشتیبانی از چند زبان**: از جفت‌های زبانی زیادی پشتیبانی می‌کند
- **آسان**: نیاز به تنظیم خاصی ندارد

### معایب
- **محدودیت ترجمه**: محدودیتی بر تعداد کاراکترهای قابل ترجمه در روز وجود دارد
- **دقت ترجمه**: دقت ممکن است در مقایسه با سرویس‌های پولی دیگر کمتر باشد
- **سرعت**: تحت تأثیر شبکه است
- **پایداری**: ممکن است به دلیل محدودیت‌های استفاده ناگهانی در دسترس نباشد

### سناریوهای استفاده توصیه‌شده
- استفاده با فرکانس پایین
- وقتی می‌خواهید فوراً شروع کنید
- وقتی می‌خواهید جفت‌های زبانی متنوعی ترجمه کنید

---

## DeepL

ماژولی که از سرویس ترجمه DeepL، معروف به ترجمه‌های با کیفیت بالا، استفاده می‌کند.

### مزایا
- **دقت بالا**: ترجمه‌های طبیعی و با کیفیت بالا
- **طبقه رایگان بزرگ**: تا 500,000 کاراکتر در ماه رایگان (API رایگان)
- **سریع**: پردازش ترجمه سریع
- **پشتیبانی از واژه‌نامه**: حفظ ثبات ترجمه با واژه‌نامه‌ها

### معایب
- **نیاز به ثبت‌نام در API**: ثبت‌نام در DeepL API و تنظیم کلید API لازم است
- **محدودیت رایگان**: اگر از طبقه رایگان تجاوز کنید، ارتقا به طرح پولی لازم است
- **زبان‌های پشتیبانی‌شده**: زبان‌های پشتیبانی‌شده محدودتر از Google است

### سناریوهای استفاده توصیه‌شده
- وقتی به دنبال ترجمه‌های با کیفیت بالا هستید
- استفاده با فرکانس متوسط

---

## Google AI (Gemini)

ماژول ترجمه‌ای که از جدیدترین فناوری هوش مصنوعی Google استفاده می‌کند.

### مزایا
- **بالاترین دقت**: ترجمه‌های با کیفیت بسیار بالا که زمینه را درک می‌کنند
- **انعطاف‌پذیری**: سفارشی‌سازی دستورالعمل برای تنظیم سبک ترجمه
- **پشتیبانی از واژه‌نامه**: حفظ ثبات ترجمه با واژه‌نامه‌ها

### معایب
- **نیاز به کلید API**: باید کلید API را از Google AI Studio دریافت و تنظیم کنید
- **پرداخت بر اساس استفاده**: هزینه بر اساس استفاده (اما مبلغ کمی)
- **سرعت**: زمان پردازش طولانی‌تر چون مبتنی بر LLM است

### سناریوهای استفاده توصیه‌شده
- وقتی به دنبال ترجمه‌های با بالاترین کیفیت هستید
- وقتی سبک ترجمه سفارشی لازم است
- وقتی ترجمه‌های حساس به زمینه لازم است

---

## ChatGPT API (یا LLM محلی)

ماژول ترجمه‌ای که از ChatGPT API یا LLM محلی استفاده می‌کند.

### مزایا
- **بالاترین دقت**: ترجمه‌های با کیفیت بالا توسط مدل‌های زبانی بزرگ
- **انعطاف‌پذیری**: سفارشی‌سازی دستورالعمل برای تنظیم سبک ترجمه
- **پشتیبانی از واژه‌نامه**: حفظ ثبات ترجمه با واژه‌نامه‌ها
- **پشتیبانی از LLM محلی**: امکان استفاده از سرور LLM خودتان

### معایب
- **نیاز به کلید API**: تنظیم کلید API برای هر سرویس لازم است (به جز LLM محلی)
- **پرداخت بر اساس استفاده**: هزینه بر اساس استفاده (به جز LLM محلی)
- **سرعت**: زمان پردازش طولانی‌تر
- **الزامات LLM محلی**: رایانه با عملکرد بالا برای اجرای LLM خودتان لازم است

### سناریوهای استفاده توصیه‌شده
- وقتی به دنبال ترجمه‌های با بالاترین کیفیت هستید
- وقتی سبک ترجمه سفارشی لازم است
- وقتی می‌خواهید ترجمه‌های با کیفیت بالا با رعایت حریم خصوصی (LLM محلی)

---

## PLaMo

ماژول ترجمه‌ای که از LLM محلی تخصصی در زبان ژاپنی استفاده می‌کند.

### مزایا
- **تخصصی در ژاپنی**: بهینه‌شده برای ترجمه‌های ژاپنی
- **کاملاً رایگان**: مدل متن‌باز بدون هزینه
- **حریم خصوصی**: به صورت محلی اجرا می‌شود، داده‌ها به خارج ارسال نمی‌شوند
- **آفلاین**: نیاز به اتصال به اینترنت ندارد

### معایب
- **الزامات عملکرد بالا**: رایانه با عملکرد بالا با GPU لازم است
- **استفاده از حافظه**: مقدار زیادی حافظه لازم است (8 گیگابایت یا بیشتر توصیه می‌شود)
- **سرعت**: زمان پردازش طولانی‌تر بدون GPU

### سناریوهای استفاده توصیه‌شده
- وقتی رایانه با عملکرد بالا دارید
- وقتی حریم خصوصی مهم‌ترین اولویت است
- وقتی کیفیت ترجمه ژاپنی را ارزش می‌گذارید
