## Скачать
### Версия для Microsoft Store ![Рекомендуется](https://img.shields.io/badge/Рекомендуется-brightgreen)

Установите из [Microsoft Store](https://apps.microsoft.com/detail/9pjd2fdzqxm3?referrer=appbadge&mode=direct).
Работает даже в средах, где .NET не установлен.

### Версия с установщиком

Скачайте `WindowTranslator-(версия).msi` со [страницы релизов GitHub](https://github.com/Freeesia/WindowTranslator/releases/latest) и запустите его для установки.

### Портативная версия

Скачайте zip-файл со [страницы релизов GitHub](https://github.com/Freeesia/WindowTranslator/releases/latest) и извлеките его в любую папку.
- `WindowTranslator-(версия).zip` : Требуется среда .NET
- `WindowTranslator-full-(версия).zip` : Независим от .NET

## Как использовать

### Bergamot ![По умолчанию](https://img.shields.io/badge/По_умолчанию-brightgreen)

1. Запустите `WindowTranslator.exe` и нажмите кнопку перевода.
   ![Кнопка перевода](images/translate.png)
2. Выберите окно приложения, которое хотите перевести, и нажмите кнопку "OK".
   ![Выбор окна](images/select.png)
