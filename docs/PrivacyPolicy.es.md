Esta política de privacidad define el tratamiento de la información personal y los datos de los usuarios al usar "WindowTranslator" (en adelante "la Aplicación").

## Introducción
La Aplicación se publica como código abierto en GitHub (https://github.com/Freeesia/WindowTranslator). La aplicación en sí básicamente no recopila información personal del usuario ni información de uso. Sin embargo, solo cuando el usuario elige explícitamente enviar un informe de error, recopilamos la información mínima necesaria para la resolución técnica de problemas.

## Acerca de la recopilación de información personal
### Recopilación de información por la aplicación
La Aplicación no recopila directamente información personal como información ingresada por el usuario, historial de operaciones o información del dispositivo durante el uso normal.
Sin embargo, solo cuando el usuario elige explícitamente enviar un informe de error, recopilamos la información necesaria para la resolución técnica de problemas.

### Registros de errores e informes de fallos
La Aplicación recopila la siguiente información solo cuando el usuario realiza explícitamente la operación de envío de informe de error:

- Información sobre el funcionamiento de la aplicación
- Información sobre las especificaciones de hardware del PC
- Información sobre el entorno del sistema operativo
- Circunstancias en el momento del error

Esta información se utiliza únicamente para resolver problemas técnicos y mejorar la calidad de la aplicación, y no contiene información que permita identificar a una persona.
El envío de informes de error es completamente voluntario y nunca se enviará automáticamente sin el consentimiento del usuario.

## Tratamiento de datos de usuario de Google
La Aplicación solo adquiere permisos relacionados con los datos de usuario de Google al acceder al servicio Google Apps Script, de la siguiente manera.

### Datos accedidos
La Aplicación solo adquiere los tokens de autenticación mínimos necesarios para usar Google Apps Script.
No se utiliza ninguna información personal del usuario.

### Cómo se utilizan los datos
La información de autenticación obtenida se utiliza únicamente para ejecutar Google Apps Script dentro de la aplicación.
Esta autenticación está limitada a la autenticación para que el usuario use los servicios de Google y no se utiliza para otros fines.

### Compartir y transferir datos
La información de autenticación y otros datos de usuario de Google no son recopilados, almacenados, compartidos ni transferidos a terceros por el proveedor de la Aplicación.
Toda la información de autenticación se almacena en el PC del usuario y solo se utiliza para los servicios de Google.

### Mecanismo de protección de datos
La información de autenticación se almacena de forma segura en el PC del usuario, y el proveedor de la Aplicación no tiene ningún derecho de acceso a esta información.
Por lo tanto, las medidas de seguridad adicionales para la protección de datos dependen de la gestión de seguridad del propio PC del usuario.

### Retención y eliminación de datos
La información de autenticación (tokens de autenticación de Google, etc.) se almacena únicamente en el PC del usuario.
La eliminación de la información de autenticación la realiza el usuario eliminando la carpeta `%APPDATA%\StudioFreesia\WindowTranslator\GoogleAppsScriptPlugin` en el PC, y no es gestionada ni eliminada por el proveedor de la Aplicación.

## Acerca del uso de motores de traducción y bibliotecas de terceros
La Aplicación está diseñada para poder utilizar múltiples motores de traducción como Google Translate, DeepL, etc. Estos motores de traducción y bibliotecas asociadas pueden recopilar información del usuario e información de uso de acuerdo con las políticas de privacidad de cada servicio.
Recomendamos encarecidamente a los usuarios que también consulten las políticas de privacidad de cada motor de traducción utilizado con la Aplicación.
